# 控制快捷键

## 全局快捷键

### 中断控制
- **ESC**：中断当前操作
- **Ctrl+C**：终止程序
- **Ctrl+Z**：挂起程序

### 导航控制
- **上下箭头**：浏览历史命令
- **左右箭头**：移动光标
- **Home/End**：跳转到行首/行尾
- **Page Up/Down**：翻页浏览

### 编辑控制
- **Backspace**：删除前一个字符
- **Delete**：删除后一个字符
- **Ctrl+U**：删除到行首
- **Ctrl+K**：删除到行尾
- **Ctrl+W**：删除前一个单词
- **Ctrl+Y**：粘贴剪贴板内容

## 快捷键提示设计

### 底部提示栏
```
─────────────────────────────────────────────
esc to interrupt | ctrl+c to exit | ? for help
```

### 提示内容
- **主要提示**：最常用的操作
- **次要提示**：次常用的操作
- **帮助提示**：获取帮助的方式

### 提示更新
- **上下文相关**：根据当前状态显示相关提示
- **动态更新**：根据操作动态更新提示
- **优先级排序**：按重要性排序提示

## 快捷键分组

### 文件操作
- **Ctrl+S**：保存
- **Ctrl+O**：打开
- **Ctrl+N**：新建
- **Ctrl+Q**：退出

### 编辑操作
- **Ctrl+C**：复制
- **Ctrl+V**：粘贴
- **Ctrl+X**：剪切
- **Ctrl+Z**：撤销
- **Ctrl+Y**：重做

### 视图操作
- **Ctrl++**：放大
- **Ctrl+-**：缩小
- **Ctrl+0**：重置缩放
- **F11**：全屏

### 搜索操作
- **Ctrl+F**：查找
- **Ctrl+H**：替换
- **Ctrl+G**：跳转到行
- **F3**：查找下一个

## 快捷键自定义

### 配置格式
```json
{
  "keybindings": {
    "interrupt": "Escape",
    "exit": "Ctrl+C",
    "help": "?",
    "history-up": "Up",
    "history-down": "Down",
    "cursor-left": "Left",
    "cursor-right": "Right"
  }
}
```

### 配置层级
1. **默认配置**：系统默认快捷键
2. **用户配置**：用户自定义快捷键
3. **项目配置**：项目特定快捷键
4. **会话配置**：会话临时快捷键

### 配置冲突处理
- **优先级**：高优先级配置覆盖低优先级
- **合并策略**：相同操作合并配置
- **冲突提示**：提示配置冲突

## 快捷键帮助系统

### 帮助显示
```
快捷键帮助:
─────────────────────────────────────────
ESC          中断当前操作
Ctrl+C       终止程序
?            显示帮助
上下箭头      浏览历史命令
左右箭头      移动光标
```

### 帮助触发
- **?**：显示快捷键帮助
- **F1**：显示完整帮助
- **Ctrl+?**：显示上下文帮助

### 帮助格式
- **分类显示**：按功能分类显示
- **表格对齐**：快捷键和说明对齐
- **搜索过滤**：支持搜索过滤快捷键

## 快捷键冲突处理

### 冲突检测
- **系统快捷键**：检测与系统快捷键冲突
- **终端快捷键**：检测与终端快捷键冲突
- **应用快捷键**：检测与其他应用快捷键冲突

### 冲突解决
- **优先级**：应用快捷键 > 终端快捷键 > 系统快捷键
- **覆盖策略**：高优先级覆盖低优先级
- **用户选择**：让用户选择冲突解决方案

### 冲突提示
```
快捷键冲突:
Ctrl+C 已被系统占用
是否覆盖?
1. 覆盖
2. 跳过
3. 修改快捷键
> _
```
