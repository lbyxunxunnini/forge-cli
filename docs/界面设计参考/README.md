## 关键设计原则

1. **符号化视觉语言** - 用简单符号传递复杂信息（`>` 输入、`✓` 成功、`*` 系统）
2. **渐进式信息披露** - 分步提问，一次一个字段，降低认知负担
3. **异步任务透明度** - 颜色 + 计时器 + 文本描述，三重机制缓解等待焦虑
4. **色彩语义化** - 绿色=正常、黄色=处理中、灰色=辅助信息
5. **紧凑信息密度** - 纯文本排版，无多余图形元素

## 源码技术栈

- **渲染引擎**: Ink (React for CLI)
- **状态管理**: React Context + useSyncExternalStore
- **布局系统**: Yoga (Flexbox for terminals)
- **主题系统**: 自定义 ThemeProvider + chalk
- **快捷键系统**: 自定义 KeybindingProvider
- **虚拟滚动**: 自定义 VirtualMessageList
