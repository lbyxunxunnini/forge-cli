# forge-cli 适配方案

## 整体架构适配

### 界面层次设计
```
┌─────────────────────────────────────────────┐
│  上部：项目信息区                              │
│  - 项目名称、版本、状态                        │
├─────────────────────────────────────────────┤
│  中部：对话与状态区                            │
│  - 用户消息、AI 响应、状态显示                 │
├─────────────────────────────────────────────┤
│  下部：输入与控制区                            │
│  - 输入提示符、快捷键提示                      │
└─────────────────────────────────────────────┘
```

### 扩展区域设计
- **左侧边栏**：可选的导航面板
- **右侧边栏**：可选的信息面板
- **底部状态栏**：系统状态和快捷键提示

## 状态显示适配

### 任务状态类型
```
* 构思中... (5s · analyzing requirements)
* 生成中... (15s · writing code)
* 优化中... (30s · refactoring)
* 测试中... (45s · running tests)
* 部署中... (60s · deploying)
```

### 工具调用状态
```
* 调用工具... (2s · reading file)
* 执行命令... (5s · running npm install)
* 访问 API... (3s · fetching data)
```

### 插件状态
```
* 加载插件... (1s · initializing)
* 执行插件... (3s · processing)
* 完成插件... (0.5s · finalizing)
```

## 快捷键适配

### 核心快捷键
- **Ctrl+C**：中断当前操作
- **Ctrl+D**：退出程序
- **Ctrl+L**：清屏
- **Ctrl+R**：搜索历史

### 工具快捷键
- **Ctrl+T**：打开工具面板
- **Ctrl+P**：打开插件管理
- **Ctrl+E**：编辑配置
- **Ctrl+L**：查看日志

### 导航快捷键
- **Tab**：切换面板
- **Shift+Tab**：反向切换面板
- **Ctrl+1-9**：切换到指定面板
- **Esc**：返回上级
