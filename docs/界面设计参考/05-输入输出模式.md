# 输入输出模式

## 输入提示符设计

### 提示符格式
```
> _
│ │
│ └── 光标（绿色闪烁）
└── 提示符符号
```

### 提示符特性
- **符号**：`>` 符号表示输入点
- **颜色**：亮绿色，高可见度
- **光标**：绿色闪烁块状光标
- **位置**：行首，左对齐

### 提示符状态
```
等待输入 → 绿色提示符 + 闪烁光标
输入中   → 绿色提示符 + 输入文本
输入完成 → 绿色提示符 + 确认显示
```

## 输入区域设计

### 输入区域布局
```
┌─────────────────────────────────────┐
│ 历史内容区域                          │
├─────────────────────────────────────┤
│ 分隔线（灰色水平线）                   │
├─────────────────────────────────────┤
│ 输入区域                             │
│ > 用户输入内容_                       │
├─────────────────────────────────────┤
│ 辅助提示区域                          │
│ esc to interrupt                    │
└─────────────────────────────────────┘
```

### 输入区域功能
1. **文本输入**：支持多行文本输入
2. **历史记录**：支持上下箭头浏览历史
3. **自动补全**：支持命令和路径补全
4. **语法高亮**：支持输入内容高亮

## 输出显示模式

### 输出类型
1. **系统消息**：灰色显示
2. **用户消息**：灰色背景 + 白色文字
3. **AI 消息**：默认前景色
4. **错误消息**：红色显示
5. **成功消息**：绿色显示

### 输出格式
```
* 系统消息
┌─────────────────────────────────────┐
│ 用户消息内容                          │
└─────────────────────────────────────┘
AI 消息内容
✓ 成功消息
✗ 错误消息
```

### 输出特性
- **自动滚动**：新输出自动滚动到底部
- **分页显示**：长输出支持分页
- **输出缓冲**：支持输出重定向
- **输出过滤**：支持按类型过滤输出

## 交互反馈机制

### 即时反馈
- **输入回显**：立即显示输入内容
- **状态更新**：实时更新处理状态
- **进度显示**：显示处理进度

### 延迟反馈
- **处理中**：显示处理状态和计时器
- **完成确认**：显示成功/失败消息
- **结果展示**：显示处理结果

### 错误反馈
- **错误定位**：指出错误位置
- **错误描述**：说明错误原因
- **修复建议**：提供修复建议
