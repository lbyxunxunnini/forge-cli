## 组件设计模式

### 1. Provider 模式
```typescript
// 主题提供者
<ThemeProvider theme={theme}>
  <App />
</ThemeProvider>

// 快捷键提供者
<KeybindingProvider>
  <App />
</KeybindingProvider>

// 向导提供者
<WizardProvider steps={steps}>
  <WizardContent />
</WizardProvider>
```

### 2. Hook 模式
```typescript
// 使用主题
const [themeName] = useTheme()

// 使用快捷键
useKeybinding('ctrl+c', () => handleCancel())

// 使用终端尺寸
const { columns, rows } = useTerminalSize()
```

### 3. 组件组合模式
```typescript
// 消息组件组合
<Message>
  <MessageResponse>
    <Markdown content={text} />
  </MessageResponse>
</Message>

// 布局组合
<FullscreenLayout
  scrollable={<Messages />}
  bottom={<PromptInput />}
  modal={<Dialog />}
/>
```

## 性能优化策略

### 1. 虚拟滚动
- **VirtualMessageList** - 只渲染可见区域的消息
- **ScrollBox** - 自定义滚动容器

### 2. 代码分割
- **feature()** - 编译时特性标志
- **动态导入** - 按需加载组件

### 3. 状态优化
- **useSyncExternalStore** - 外部状态同步
- **useDeferredValue** - 延迟更新
- **React.memo** - 组件缓存

### 4. 渲染优化
- **React Compiler** - 自动优化渲染
- **批处理** - 批量更新状态
- **requestAnimationFrame** - 动画帧控制
